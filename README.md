# Borges Silva Locações

Sistema interno de gestão de locações.
Suporta imóveis residenciais (casas, apartamentos) e unidades comerciais do Center Lila.

## Stack
- Next.js 14 (App Router) + TypeScript
- Supabase (PostgreSQL + Auth + Storage)
- Tailwind CSS + shadcn/ui
- Vercel (deploy)

## Configuração
1. Copie `.env.example` para `.env.local` e preencha as chaves do Supabase
2. `npm install && npm run dev`
